## 关于 Hexo

Hexo 是一个快速、简洁且高效的博客框架。它使用 Markdown 语法解析文章，在几秒内即可生成静态页面。

### 特点

- ⚡ **极速生成** — 秒级生成静态页面
- 📝 **Markdown 支持** — 用 Markdown 写文章，语法简单
- 🎨 **丰富主题** — 数百款精美主题可供选择
- 🔧 **可扩展** — 支持插件扩展功能

### 写作示例

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello Hexo!");
    }
}
```
