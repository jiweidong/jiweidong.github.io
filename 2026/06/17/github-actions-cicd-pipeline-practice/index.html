GitHub Actions CI/CD 实战：从入门到生产级流水线

一、GitHub Actions 核心概念

GitHub Actions 是 GitHub 原生的 CI/CD 工具，让代码托管和自动化流水线无缝集成。与其他 CI/CD 工具相比，它的最大优势是与 GitHub 仓库的深度集成。

5.1 什么时候用自托管 Runner

需要自托管 Runner 的场景：

  • 需要访问内网资源（数据库、K8s 集群）
  • 需要 GPU 或特定硬件
  • Maven/NPM 缓存放在内网
  • 免费额度不够用
